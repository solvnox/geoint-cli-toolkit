# solvnox GEOINT / OSINT CLI

Интерактивная утилита геопространственной разведки (GEOINT/OSINT) с меню на русском языке  

## Установка

```bash
cd geoint_cli
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```
